# mybokji

> 모든 복지를 놓치지않고 추천해주는 앱

## 1. 제품 비전

### 해결하려는 문제
사용자들이 자신에게 적합한 복지 혜택을 알지 못하거나 놓치는 문제를 해결합니다. 복잡한 정보 탐색 과정을 간소화하여 개인화된 복지 정보를 제공합니다.

### 목표 사용자
- 복지 혜택을 잘 모르는 일반 시민
- 복지 정보 접근이 어려운 취약 계층
- 복지 관련 정보를 제공해야 하는 사회복지사

### 핵심 가치
1. **개인화된 추천**: 사용자의 상황에 맞는 복지 혜택을 자동으로 추천합니다.
2. **정보 접근성 향상**: 복잡한 복지 정보를 간단하고 명확하게 제공합니다.
3. **시간 절약**: 사용자가 필요한 정보를 빠르게 찾을 수 있도록 돕습니다.

## 2. 주요 기능

| 기능명                | 설명                                                                 | 우선순위 |
|-----------------------|----------------------------------------------------------------------|----------|
| user-profile          | 사용자의 기본 정보(나이, 소득, 거주지 등)를 입력받아 저장합니다.      | 높음     |
| welfare-recommendation| 입력된 정보를 기반으로 개인화된 복지 혜택을 추천합니다.               | 높음     |
| search-welfare        | 키워드로 복지 혜택을 검색할 수 있습니다.                              | 중간     |
| save-favorites        | 관심 있는 복지 혜택을 저장하고 관리할 수 있습니다.                    | 중간     |
| notification-system   | 새로운 복지 혜택이 추가되면 알림을 보냅니다.                          | 낮음     |
| admin-dashboard       | 관리자가 복지 데이터를 추가/수정/삭제할 수 있는 대시보드를 제공합니다. | 낮음     |
| analytics-report      | 사용자의 복지 이용 통계를 시각화하여 제공합니다.                      | 낮음     |

## 3. 추천 기술 스택

- **프론트엔드**: React, TypeScript, Tailwind CSS
- **백엔드**: Node.js, Express
- **데이터베이스**: PostgreSQL
- **인프라/배포**: AWS (EC2, S3, RDS), Docker
- **기타**: Jest (테스트), Swagger (API 문서화)
